# SR Flip-Flop

## Theory
An SR Flip-Flop is a sequential circuit with Set (S) and Reset (R) inputs used to store one bit of data.

## Truth Table

| S | R | Q(next) |
|---|---|---------|
| 0 | 0 | Q(prev) |
| 0 | 1 | 0 |
| 1 | 0 | 1 |
| 1 | 1 | Invalid |
