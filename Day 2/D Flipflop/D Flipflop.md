
# D Flip Flop

## Theory
A D Flip-Flop is a sequential circuit that stores one bit of data. The output Q follows the input D on the active clock edge.

## Truth Table

| D | Q(next) | Q' |
|---|----------|----|
| 0 | 0 | 1 |
| 1 | 1 | 0 |
